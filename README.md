## 這版先測什麼

- 月曆任務地圖：一次看一個月，點日期切換任務
- 每月願望：每個月解鎖一個小目標
- 今日任務打卡：手機可拍照，電腦可選照片
- 完成前必須加入照片
- 完成基礎任務後才解鎖進階感受任務
- 年度成長書門檻：240 天基礎任務、180 次有效照片打卡、120 篇合格進階感受、9 個月徽章
